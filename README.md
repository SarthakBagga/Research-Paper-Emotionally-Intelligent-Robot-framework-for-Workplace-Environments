# Research-Paper-Emotionally-Intelligent-Robot-framework-for-Workplace-Environments
Worked with Dr. Armaan Garg and designed and developed an emotionally and socially aware robotic system aimed at improving human-robot interaction in academic environments.


This project focused on building a privacy-aware, adaptive robot capable of multi-modal emotion recognition and context-sensitive behavior. My contributions spanned across system architecture design, perception, and behavior modeling. We implemented a multi-modal emotion recognition pipeline using visual, posture, and speech cues, and integrated it with a context-aware policy that adapts robot behavior based on user roles and environmental conditions. Additionally, We designed a consent-driven interaction framework with explainable decision-making to address trust and privacy concerns. The system was validated through a participatory user study, where we analyzed feedback to refine behavior policies and improve system acceptance. This work reflects my interest in combining robotics, human-centered AI, and reinforcement learning to build socially intelligent autonomous systems.
